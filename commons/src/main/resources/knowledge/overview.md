# 国内自由行通用指南

本知识库面向全国范围内的城市自由行与周边短途游，供行程规划 Agent 检索参考。

## 规划原则

- 每日安排 2-4 个核心点位，留足交通与用餐缓冲。
- 热门城市周末与法定节假日需预留排队时间。
- 亲子、老人同行时减少换乘次数，优先选择地铁直达或打车成本可控区域。

## 交通建议

- 抵达陌生城市优先确认机场/高铁站至市区的末班车时间。
- 地铁+步行通常是一日游最高效组合；远郊景点需单独规划往返时长。
- 自驾注意景区停车容量、限行政策及山区弯道行车安全。

## 住宿选择

- 首次到访建议住在地铁沿线的市中心或核心商圈，便于串联景点。
- 古城类目的地（如丽江、平遥）可考虑住古城内，但注意行李搬运与噪音。
- 预订前查看近期评价中的「卫生」「隔音」「实际步行至地铁距离」。

## 用餐提示

- 网红店适合体验，本地日常餐饮可询问酒店前台或查看商场 B1 层。
- 景区周边价格偏高，可选择在景区外 1-2 公里就餐。
- 西北、川渝等地口味偏重，可提前说明忌口。

## 季节与装备

- 夏季：防晒、补水、防蚊；户外景点避开 11:00-15:00 暴晒时段。
- 冬季：北方注意防风保暖，南方湿冷需准备防潮外套。
- 高原地区（如拉萨、香格里拉）需关注高反，首日避免剧烈运动。
